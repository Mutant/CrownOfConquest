Changelog

0.2.2 - 28/03/2009

Features

  • Main map can now be zoomed out
  • The sage can now find dungeons

Bug Fixes & Minor Additions

  • List of known dungeons are now displayed on main map page
  • Max durability of an item is reducted during upgrades at the blacksmith
  • Other parties will now be displayed in dungeons
  • Fixed a bug with count of creatures in dungeons
  • The level of the dungeon is now displayed
  • Fixed a bug that could cause infinite loops (and kill the server) during combat

0.2.1 - 08/03/2009

Features

  • Some items now have a durability stat, which degrades with use. They can be reapaired at the blacksmith
  • Certain items can also be upgraded at the blacksmith

Bug Fixes & Minor Additions

  • Some optimisations made to dungeon movement
  • Creature groups can now contain more than one creature type
  • Fixed a bug where map screen wasn't always getting refreshed after combat
  • Monster groups in dungeons now display a tooltip listing the creatures in the group

0.2.0 - 22/02/2009

Features

  • Initial Release of Dungeons

0.1.6 - 23/01/2009

Features

  • Added a quest to destroy Creature Orbs.

Bug Fixes & Minor Additions

  • Creatures no longer attack higher level parties
  • Lower level parties won't get attack by Creature groups guarding orbs
  • Number of spells now displayed next to the spell name in cast menu
  • Fixed ordering of characters on Shop, etc. screens
  • Made messages more descriptive on the Quest list
  • Fixed a bug that caused multiple creature groups to spawn in the same sector

0.1.5 - 21/01/2009

Features

  • Added "Creature Orbs", which spawn creatures. These can be destroyed by your party.

Bug Fixes & Minor Additions

  • Fixed some bugs that prevent creatures spawning
  • Leaderboard now displays XP per Turn
  • Added a game stats page
  • Fixed some problems around groups of creatures that're all dead
  • Fixed a bug that prevented editing of characters during party creation after the fifth character had been created
  • Fixed day numbers on graves, which were being set incorrectly
  • Added login info and list of parties on left hand nav
  • Removed the 'undefined' text from the dialog when selling a jewel for a jewel quest
  • Fixed issue with count of characters on the leaderboard

0.1.4 - 13/01/2009

Features

  • Dead Characters can now be buried, with an Epitaph

Bug Fixes & Minor Additions

  • Dead creatures are no longer displayed in creature group count
  • Can now change character's names
  • Can no longer sell dead characters
  • Fixed some issues with the news - this may mean some news items don't appear in the short term
  • Improved the Spells tab on the character view
  • Town's prosperity now shown in Known Towns on map screen
  • Can now edit and delete character during party creation
  • New characters now start with some basic equipment
  • Fixed some problems around party order and the rank separator when selling a character

Features

Bug Fixes & Minor Additions

  • Fixed a bug that let you sell a Jewel many times
  • Attempted to fix some issues around caching
  • Fixed the Town News, which was displaying combats from a much larger area than it should've been
  • Colour co-ordinated the combat messages

Features

Bug Fixes & Minor Additions

0.1.1 - 25/12/2008

Features

  • List of known towns now displayed on the map screen
  • You can now disband your party (option available from the party screen)
  • Added "History" tab for characters
  • Items will now be auto-equipped when purchasing them (if nothing is already equipped in that slot)

Bug Fixes & Minor Additions

  • Fixed crash bug on character's equipment tab if the character has no equipment
  • Added a warning when a player hasn't assigned all points from the stat pool when creating a new character
  • Prevented negative modifers being used when creating a character
  • Tweaked layout of party screen
  • Fixed a bug that caused an error when a creature's effect expired
  • Error messages are now displayed when trying to heal for more gold than you have
  • Fixed some bugs in the 'New Day' messages
  • Fixed a bug which caused an error when casting a spell as combat ended
  • Fixed cosmetic problem when assigning stat points
  • Now display an indicator on the main screen when a character has stat points to use
