Changelog

0.1.5 - ??/01/2009

Features

  • Added "Creature Orbs", which spawn creatures. These can be destroyed by your party.

Bug Fixes & Minor Additions

  • Fixed some bugs that prevent creatures spawning

0.1.4 - 13/01/2009

Features

  • Dead Characters can now be buried, with an Epitaph

Bug Fixes & Minor Additions

  • Dead creatures are no longer displayed in creature group count
  • Can now change character's names
  • Can no longer sell dead characters
  • Fixed some issues with the news - this may mean some news items don't appear in the short term
  • Improved the Spells tab on the character view
  • Town's prosperity now shown in Known Towns on map screen
  • Can now edit and delete character during party creation
  • New characters now start with some basic equipment
  • Fixed some problems around party order and the rank separator when selling a character

0.1.3 - 08/01/2009

Features

  • Added Recruitment Office

Bug Fixes & Minor Additions

  • Fixed a bug that let you sell a Jewel many times
  • Attempted to fix some issues around caching
  • Fixed the Town News, which was displaying combats from a much larger area than it should've been
  • Colour co-ordinated the combat messages

0.1.2 - 04/01/2009

Features

  • Added Basic Party Leaderboards

Bug Fixes & Minor Additions

  • Made layout look a bit better on lower resolutions
  • Fixed some problems around groups of dead creatures attacking

0.1.1 - 25/12/2008

Features

  • List of known towns now displayed on the map screen
  • You can now disband your party (option available from the party screen)
  • Added "History" tab for characters
  • Items will now be auto-equipped when purchasing them (if nothing is already equipped in that slot)

Bug Fixes & Minor Additions

  • Fixed crash bug on character's equipment tab if the character has no equipment
  • Added a warning when a player hasn't assigned all points from the stat pool when creating a new character
  • Prevented negative modifers being used when creating a character
  • Tweaked layout of party screen
  • Fixed a bug that caused an error when a creature's effect expired
  • Error messages are now displayed when trying to heal for more gold than you have
  • Fixed some bugs in the 'New Day' messages
  • Fixed a bug which caused an error when casting a spell as combat ended
  • Fixed cosmetic problem when assigning stat points
  • Now display an indicator on the main screen when a character has stat points to use
