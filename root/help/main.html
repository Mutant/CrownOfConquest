Kingdoms Help

For those wanting to dive right in, the Quick Start Guide is a good place to start.

The Kingdom's help is split into several sections:

  • Getting Starting is for new parties, and provides an introduction to the game and some hints on how to play.
  • Reference provides some more general information on parts of the game.
  • Miscellaneous has information about Kingdoms, the change log, and announcements about the game.
